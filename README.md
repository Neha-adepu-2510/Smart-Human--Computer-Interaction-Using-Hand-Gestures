# 🖐 Smart Human-Computer Interaction Using Hand Gestures

An AI-powered real-time hand gesture recognition system that enables touchless computer control using natural hand movements. The system uses computer vision to detect hand landmarks via webcam and maps specific gestures to system-level actions.

---

##  Overview

This project demonstrates how Human-Computer Interaction (HCI) can be enhanced using Computer Vision and AI.  
By leveraging real-time hand tracking, users can control their system without physical contact, making interaction more intuitive and hygienic.

---

##  Features

- Real-time hand detection and tracking  
- Gesture-based virtual mouse control  
- Click and scroll using finger gestures  
- Volume and brightness adjustment  
- Browser tab switching  
- Custom exit gesture  
- Smooth and responsive performance  

---

##  Installation & Setup

1. Clone the repository

```
git clone https://github.com/nehaadepu/Smart-Human--Computer-Interaction-Using-Hand-Gestures.git
```

2. Navigate to the project folder

```
cd Smart-Gesture-Control
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the project

```
python gesture_controller.py
```
